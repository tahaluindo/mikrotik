Informasi dan Tutorial

Pastikan area anda tercover dengan signal wifi kami

Aktifkan wireless pada smartphone atau laptop anda dan koneksikan dengan google kami

Tunggu beberapa saat sampai pop-up notifikasi untuk login muncul, jika tidak ada notifikasi silakan buka apps browser (chrome, mozilla) pada smartphone atau laptop anda dan silahkan ketikan manual url halaman login yang tertera pada website kami

Setelah anda berhasil masuk ke halaman login silakan aktivasi Tools dan klik Home Page

User Name salah: Notifikasi ini tampil ketika anda salah mengetikan ID API KEY harap selalu perhatikan penulisan huruf besar dan kecil

User Name sudah dipakai: Jika tampil notifikasi ini berarti ID API KEY yang anda masukan sudah dipakai oleh pengguna lain, 1 API KEY hanya bisa dipakai untuk 1 perangkat saja

  • Masa Aktif sudah habis: Peringatan ini akan tampil jika masa aktif ID API KEY anda sudah mencapai limit / habis

    • Masa Aktif Api KEY anda terhitung sejak pertama kali digunakan / login

    Jika pada ApiKey anda tertera Durasi dan Masa Aktif maka cara perhitungannya sebagai berikut, misalkan durasi 3 jam, masa aktif 1 hari artinya ApiKey tersebut bisa digunakan selama 3 jam, jika masih ada sisa waktunya, ApiKey tersebut hanya bisa digunakan maksimal 1 hari saja sejak pertama kali diaktifkan

    Jika anda masih bingung cara berlangganan atau aktivasi Api Key kami, silahkan hubungi admin layanan untuk meminta panduan penggunaan

    Jika terjadi masalah seperti gangguan jaringan, dan lain-lain tim kami akan menginformasikannya kepada anda

    Selama anda menjadi pelanggan kami, semua masalah seputar layanan internet akan kami bantu
